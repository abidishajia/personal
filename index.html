Who Am I?

My interest in web development started in my last semester at San Francisco State University when I took the Data Journalism class. The class required me to learn HTML/CSS, JavaScript and PHP on codeacademy.com. After learning the basics from there, I created this webpage which can be used to filter down 445 school districts in San Francisco County by districts, CCSF Entitiy and type.

The class was limited to basics, so I enrolled in Udacity's Nanodegree Program: Front-End Web Development. From this program, I learned a whole lot about responsive design, Grunt, GitHub, Javascript's libraries, Web Optimization, Google API's and much more.

Now I can create user-friendly resposive sites. My work can be found on my GitHub page.

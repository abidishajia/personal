Who Am I?

One Thursday evening while I was trying to figure out how to implement Bootstrap on my web page, I realized I wasn't giving up. I tried. I failed. I tried again. I failed again. It was frustrating, but I kept trying until I came up with the solution.

On that day in my Data Journalism class, I knew I wanted to write code for the rest of my life instead of hard news.

I pursued my interest and enrolled in a Nanodegree Front-End Web Development course at Udacity where I developed the skills required to become a Front-End Web Developer; I became proficient with HTML, CSS and JavaScript. I learned how to build responsive websites, create games using Object Oriented JavaScript, use Google APIs, Grunt and web optimization on my page, and how to implement JavaScript libraries and frameworks.

My projects can be found on my GitHub Account at abidishajia .

San Francisco School Map
